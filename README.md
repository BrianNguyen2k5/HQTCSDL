Nhóm đồ án của môn HQTCSDL

Hướng dẫn cách cài Tài khoản cho nhân viên.

B1: Chạy tất cả các file sql trong folder database.

B2: Chạy lệnh cd để đi vào thư mục "../HQTCSDL/UpdatePasswordTool/'

B3: chạy lệnh dotnet run

Sau khi chạy dotnet run, chương trình sẽ trả về kết quả cài tài khoản thành công Các tài khoản đăng nhập của từng vai trò nhân viên như sau:

--CV01 Quản lý => NV001 - Username: nv_NV001, pass: 123456

--CV02 Lễ tân => NV006 - Username: nv_NV006, pass: 123456

--CV03 Kỹ thuật => NV002 - Username: nv_NV002, pass: 123456

--CV04 Thu ngân => NV003 -Username: nv_NV003, pass: 123456

Còn phía khách hàng, có để thực hiện đăng ký để tạo tài khoản và đăng nhập bình thường.
